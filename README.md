# Satisfação funcionarios
Este é um projeto de ciencia de dados e machine learning.

O dataset usado contem dados que contribuem para a satisfação ou dessatisfação de funcionarios de uma empresa.

Quanto maior a dessastifação de funcionarios em certas variaveis, maior a chance da demissão voluntaria.

No final do projeto chegamos a conclusão de que os fatores que contribuem para a demissão voluntaria de funcionarios são:

- Viagens de trabalhos frequentes
- Horas extras
- E pessoas solteiras

Foi usado um modelo de Regressão Logistica para chegar neste resultado, com uma acuracia de 85%

O projeto também contem dois graficos com analise exploratoria de variaveis do dataset.
