# Satisfação funcionarios
Este é um projeto de ciencia de dados e machine learning.

O dataset usado contem dados que contribuem para a satisfação ou dessatisfação de funcionarios de uma empresa.

quanto maior a dessastifação de funcionarios em certas variaveis, maior a chance da demissão voluntaria.


Fonte: https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset/data
